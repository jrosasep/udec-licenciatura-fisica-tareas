# Tarea 02 — Óptica matricial

Tarea dedicada al uso de matrices $ABCD$ para describir sistemas ópticos: superficies esféricas, lentes delgadas
y sistemas compuestos.

## Contenido

- **Situación 1 — Objeto dentro de una esfera de vidrio:**
  - Esfera maciza con índice $n' = 8/5$ y radio $R = 3\\ \\text{cm}$.
  - A partir de la posición aparente del objeto se determina la posición real usando la matriz de refracción
    en superficie esférica.
  - Cálculo de distancias focales, posiciones de imagen y puntos cardinales asociados.

- **Situación 2 — Lente delgada convergente:**
  - Lente de focal $f = 15\\ \\text{cm}$ con un objeto a $5\\ \\text{cm}$.
  - Uso de la matriz de lente delgada para encontrar distancia imagen, aumento lateral y tipo de imagen
    (virtual, derecha y aumentada).
  - Construcción cualitativa del diagrama de rayos.

- **Situación 3 — Sistema de dos lentes:**
  - Lente convergente seguida de lente divergente separadas $60\\ \\text{cm}$.
  - Cálculo de la matriz total del sistema, focos equivalentes y puntos cardinales.
  - Ubicación de la imagen final, aumento resultante y diagrama de rayos.

- **Problema adicional — Placa SELFOC:**
  - Medio con índice cuadrático $n^2(y) = n_0^2(1 - \\alpha^2 y^2)$.
  - Demostración de que actúa como lente cilíndrica con longitud focal $f_2 = 1/(\\alpha \\sin(\\alpha d))$.
