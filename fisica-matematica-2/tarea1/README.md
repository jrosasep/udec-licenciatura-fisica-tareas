# Tarea 01 – Transformadas Integrales

Tarea 01 de Física Matemática 2, enfocada en series y transformadas de Fourier.

## Ejercicios

- **Ejercicio 1:** función triangular definida en el intervalo $[0,3]$. Se obtiene la expresión a trozos, se extiende de forma periódica a todo ℝ y se calculan la serie de Fourier trigonométrica, la extensión impar y la serie de Fourier exponencial asociada.
- **Ejercicio 2:** pulso triangular del tipo $f(t) = 1 - a|t|$ para $|t| < 1/a$ y cero fuera de ese intervalo. Se calcula su transformada de Fourier usando la versión coseno, cuidando bien las condiciones de paridad.
